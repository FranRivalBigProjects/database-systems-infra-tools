❓ Frequently Asked Questions

What is database sharding?

Sharding splits data across multiple servers to improve scalability.

What is replication?

Replication copies data across nodes for redundancy and availability.

Why is indexing important?

Indexing improves query speed by reducing the amount of data scanned.

What is connection pooling?

It reuses database connections to improve performance and reduce overhead.
